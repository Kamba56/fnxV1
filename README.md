# FNX Webapp

## Built with
- Vite react
- Tailwindcss for styling
- React icons
- AOS for animation

## Getting started
### prerequisites
- nodejs
- git

to get a local copy running;
- clone the repository
```git clone https://github.com/Kamba56/fnxV1.git```
- change the directory and install dependencies
```cd fnxV1```
```npm install```
- run on local server
```npm run dev```
